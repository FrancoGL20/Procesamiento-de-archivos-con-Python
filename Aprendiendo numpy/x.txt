**Pandas** es una biblioteca de análisis de datos de código abierto para el lenguaje de programación **Python**. Fue creada por Wes McKinney en 2008 y se ha convertido en una herramienta popular para el análisis y manipulación de datos. La biblioteca proporciona estructuras de datos flexibles y de alto rendimiento diseñadas para facilitar el trabajo con datos estructurados, como hojas de cálculo, tablas SQL o bases de datos NoSQL. 

Pandas es una herramienta muy útil para la limpieza, transformación y análisis de datos. Algunas de las características que ofrece son:

- Lectura y escritura de datos en diferentes formatos: CSV, Excel, SQL, JSON, HTML, entre otros.
- Selección, filtrado y transformación de datos.
- Combinación y unión de conjuntos de datos.
- Manejo de valores faltantes.
- Agrupación y agregación de datos.
- Visualización de datos.
